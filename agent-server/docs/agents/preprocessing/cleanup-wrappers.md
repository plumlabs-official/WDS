# Cleanup Wrappers (의미 없는 래퍼 제거)

## 개요
단일 자식만 가진 프레임을 제거하여 레이어 구조를 단순화합니다.

## 타입
- **Rule-based** (LLM 사용 안 함)

## 삭제 기준

### 삭제 대상
1. 자식이 1개뿐인 프레임
2. Auto Layout이 적용되지 않은 프레임
3. 특별한 스타일(배경색, 테두리 등)이 없는 프레임

### 삭제하지 않는 경우
- 자식이 0개 또는 2개 이상
- Auto Layout이 적용된 프레임
- 배경색, 테두리, 그림자 등 시각적 스타일이 있는 프레임
- 클리핑(clipsContent)이 활성화된 프레임
- 컴포넌트 또는 인스턴스

## 동작 방식
1. 선택된 프레임의 모든 자식을 재귀적으로 탐색
2. 삭제 기준에 맞는 래퍼 프레임 식별
3. 래퍼의 자식을 부모로 이동 (위치 보존)
4. 래퍼 프레임 삭제

## 주의사항
- 삭제된 노드에 접근하면 에러 발생 → `isNodeValid()` 체크 필수
- 자식 이동 시 좌표 변환 필요 (로컬 → 부모 좌표계)
